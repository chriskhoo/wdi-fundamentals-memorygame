Instructions

Concentration, also known as Match Match, Memory, Pelmanism, Shinkei-suijaku,Pexeso or simply Pairs, is a card game in which all of the cards are laid face down on a surface and two cards are flipped face up over each turn. The object of the game is to turn over pairs of matching cards.

Why is it awesome?

The memory game is suitable for people of all ages. Playing memory games not only trains your visual memory, but has been medically proven to help overall brain function. The benefits include:

  • Improved concentration
  • Improved focus
  • Enhanced cognitive skills
  • Slows onset of memory related illness
